Usage Guide

컬러 사용 가이드

배경 (Background)

메인: #000000 (Pure Black)
카드: #141414 (Light Black)
섹션: #1A1A1A (Muted Black)

카드 (Light Mode)

배경: #F5F2EE → #E8E3DD
호버: #E0DBD5 → #FFF
보더: #B5A999

텍스트 (Typography)

헤드라인: #FFFFFF
본문: #E5E5E5
서브: #737373
⚠️ 베이지 사용 금지

CTA / 강조

Primary: #FFFFFF
Secondary: #C4B8A8 outline
Card: #F5F2EE 배경

CSS Variables

복사하여 사용할 수 있는 CSS 변수

:root {
  /* Base Colors - Black */
  --black: #000000;
  --black-soft: #0A0A0A;
  --black-light: #141414;
  --black-muted: #1A1A1A;

  /* Base Colors - White (텍스트용) */
  --white: #FFFFFF;
  --white-muted: #E5E5E5;

  /* Logo Beige (배경/카드용) - 텍스트 사용 금지 */
  --beige: #C4B8A8;
  --beige-light: #F5F2EE;
  --beige-soft: #E8E3DD;
  --beige-hover: #E0DBD5;
  --beige-dark: #B5A999;

  /* Gray Scale */
  --gray-500: #737373;
  --gray-600: #525252;
  --gray-700: #404040;
  --gray-800: #262626;

  /* Gradients */
  --gradient-beige: linear-gradient(135deg, #F5F2EE 0%, #E8E3DD 100%);
  --gradient-beige-hover: linear-gradient(135deg, #E0DBD5 0%, #FFFFFF 100%);
  --gradient-dark: linear-gradient(135deg, #000000 0%, #1A1A1A 100%);
}

# IBN 웹사이트 스타일 가이드

> 모든 페이지 작업 시 적용하는 프로젝트 가이드입니다.

---

## 1. 절대 수정 금지 영역

### 공통 컴포넌트 (모든 페이지에서 유지)

| 컴포넌트 | 설명 | 비고 |
|----------|------|------|
| **헤더** | 상단 네비게이션 | 로고, 메뉴, CTA 버튼 |
| **푸터** | 하단 정보 영역 | 회사정보, 링크, 카피라이트 |
| **입력폼** | 무료상담 신청 폼 | 배지, 타이틀, 혜택, 폼 영역 |
| **모바일 네비게이션** | 하단 고정 메뉴 | 6개 메뉴 버튼 |

### 입력폼 상세 규칙

입력폼 섹션은 **완전히 통일**되어 있습니다. 절대 수정하지 마세요.

```
[입력폼 구조]
├── 배지: "IBN 정책자금 컨설팅"
├── 타이틀: "**IBN 전문가**에게 문의하세요" (검정배경 강조)
├── 혜택 리스트
│   ├── 95% 승인률
│   ├── 평균 2억원+ 지원
│   ├── 전 과정 밀착 지원
│   └── 100% 무료 상담
├── 폼 제목: "IBN 무료 상담 신청"
└── 버튼: 마우스 추적 스포트라이트 효과
```

### 버튼 스타일 (통일)

```css
/* 입력폼 버튼 - 마우스 추적 효과 */
.submit-btn {
    background: #000000;
    color: #FFFFFF;
    /* 마우스 추적 스포트라이트 효과 적용 */
}
```

## 2. 브랜드 리라이팅 규칙

### 브랜드명 변경

| 기존 표현 | 변경 후 |
|----------|--------|
| 더나은 기업연구소 | **IBN** |
| IBN | **IBN** |
| 기업심사관 시스템 | **IBN 컨설팅 시스템** 또는 **전 과정 밀착 지원** |
| 기업심사관 출신 전문가 | **정책자금 전문가** |
| 담당 기업심사관 | **담당 전문 컨설턴트** |

### 수치 통일 (중요!)

| 항목 | 통일 수치 | 비고 |
|------|----------|------|
| 승인률 | **95%** | 유지 |
| 평균 지원액 | **2억원+** | 3.2억원에서 변경 |
| 총 자금 확보액 | **100억+** | 신규 추가 |

### 주요 문구 변환 예시

```
❌ "기업심사관 출신 전문가가 직접 컨설팅"
✅ "IBN 전문팀이 직접 컨설팅"

❌ "평균 3.2억원 승인"
✅ "평균 2억원+ 지원"

❌ "더나은 기업연구소만의 시스템"
✅ "IBN만의 검증된 시스템"

❌ "담당 기업심사관이 연락드립니다"
✅ "담당 전문 컨설턴트가 연락드립니다"
```

## 3. IBN 컬러 컨셉 (검정/흰색 기반)

### 핵심 컬러

| 용도 | 컬러 | HEX | 예시 |
|------|------|-----|------|
| **Primary** | 검정 | `#000000` | 메인 텍스트, 강조 배경, 버튼 |
| **Secondary** | 흰색 | `#FFFFFF` | 배경, 버튼 텍스트 |
| **Accent** | 연한 회색 | `#E5E5E5` | 서브 요소, 구분선 |

### 텍스트 컬러 체계

| 용도 | 컬러 | HEX |
|------|------|-----|
| 메인 텍스트 | 검정 | `#000000` |
| 서브 텍스트 | 진회색 | `#4B5563` |
| 보조 텍스트 | 회색 | `#6B7280` |
| 강조 텍스트 | 흰색 (검정 배경) | `#FFFFFF` |

### 배경색

| 용도 | 컬러 |
|------|------|
| 기본 배경 | `#FFFFFF` |
| 섹션 구분 배경 | `#F8FAFB` ~ `#F5F5F5` |
| 카드 배경 | `#FFFFFF` |
| 강조 배경 | `#000000` |
| 이미지 플레이스홀더 | `#F5F5F5` → `#E5E5E5` 그라데이션 |

### 강조 텍스트 스타일 (핵심!)

**검정 배경 + 흰 폰트** 방식으로 통일

```css
/* 강조 텍스트 */
.highlight {
    color: #FFFFFF;
    background: #000000;
    padding: 2px 10px;
    border-radius: 4px;
    font-weight: 800;
}
```

### 버튼 스타일

```css
/* 기본 버튼 */
.btn-primary {
    background: #000000;
    color: #FFFFFF;
    border: 2px solid #000000;
}

.btn-primary:hover {
    background: #333333;
    border-color: #333333;
}

/* 아웃라인 버튼 */
.btn-outline {
    background: #FFFFFF;
    color: #000000;
    border: 2px solid #000000;
}

.btn-outline:hover {
    background: #000000;
    color: #FFFFFF;
}
```

## 4. 사용 금지 컬러

| 컬러 | HEX | 이유 |
|------|-----|------|
| 녹색/틸 | `#0C2E21`, `#0D9488`, `#6BA88D` | IBN 브랜드 컬러 아님 |
| 파랑 | `#1B3A5F` | 이전 브랜드 컬러 |
| 밝은 민트 | `#8FBFA8` | 히어로 배경 제외 사용 금지 |

### 히어로 섹션 예외

히어로 섹션(어두운 배경)에서만 다음 컬러 사용 가능:
- 배경: 어두운 그라데이션
- 포인트: 연한 민트 계열 (제한적)

## 6. 작업 체크리스트

### 새로운 페이지/섹션 작업 시

1. **공통 컴포넌트 확인**
   - [ ] 헤더 그대로 유지
   - [ ] 푸터 그대로 유지
   - [ ] 입력폼 그대로 유지

2. **브랜드명 검색 & 치환**
   - [ ] "더나은" → "IBN"
   - [ ] "IBN" → "IBN"
   - [ ] "기업심사관" → 적절한 대체 표현

3. **수치 확인**
   - [ ] 승인률 95% 확인
   - [ ] 평균 지원액 2억원+ 확인
   - [ ] 총 확보액 100억+ 확인

4. **컬러 확인**
   - [ ] 녹색/틸 컬러 사용 안 함
   - [ ] 파랑 계열 사용 안 함
   - [ ] 검정/흰색/회색 계열만 사용
   - [ ] 강조 텍스트 = 검정 배경 + 흰 폰트

5. **반응형 확인**
   - [ ] 모바일 폰트 사이즈 적절한지
   - [ ] 줄바꿈 정상 작동하는지
   - [ ] 터치 타겟 44px 이상인지

## 7. 주의사항

### 흰 배경 + 흰 폰트 문제

```
⚠️ 주의: 흰 배경 섹션에서 흰색 텍스트 사용 금지!

문제 발생 케이스:
- .highlight { color: #FFFFFF; } + 흰 배경 = 안 보임

해결:
- 강조 텍스트는 반드시 검정 배경과 함께 사용
- 또는 검정/진회색 텍스트로 변경
```

### 줄바꿈 처리

```css
/* 모바일에서 줄바꿈 적용 */
.text-container {
    display: block;  /* flex에서 block으로 변경 */
}

/* 또는 HTML에서 <br class="mobile-br"> 태그 사용 */
```
